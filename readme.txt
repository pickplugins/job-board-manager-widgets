== Description ==

"Job Board Manager - Widgets" allows you to display some useful Widgets  for "Job Board Manager" plugin.


### Job Board Manager - Widgets by http://pickplugins.com
* [Live Demo (Sidebar) &raquo;](http://www.pickplugins.com/demo/job-board-manager/)
* [Get Job Board Manager &raquo;](https://wordpress.org/plugins/job-board-manager/)

# Widgets:

* Job Board Manager - Expired Today
* Job Board Manager - Featured Job
* Job Board Manager - Latest Job


<strong>Translation</strong>

Plugin is translation ready, you can contribute in translation. please contact with us http://pickplugins.com/contact/

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Job Board Manager - Widgets</strong>" activate it.<br />
